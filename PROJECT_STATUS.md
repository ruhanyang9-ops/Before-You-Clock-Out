# 项目进度与未完成工作

更新时间：2026-06-11

## 当前快照

- 本地访问地址：`http://127.0.0.1:4173/`
- 数据库：`data/auth.sqlite` 保存账号；`data/users/*.sqlite` 保存每个用户的个人业务数据；旧 `data/app.sqlite` 会由首个注册用户继承。
- 当前本地旧数据库已有数据会在首个注册账号时迁移到该账号的个人数据库。
- AI 状态：已支持 DeepSeek 真实 AI；未配置或调用失败时自动 fallback 到本地 mock。
- 语音状态：已支持百度智能云后端语音转文字；浏览器语音识别不可用时可回退后端录音转写，转写后会自动润色预处理。
- 语音时长：单次输入最长 3 分钟，界面已标识，到时自动停止并处理。

## 文件整理结论

当前项目文件数量少，且 `index.html`、`styles.css`、`app.js`、`server.js` 默认以根目录静态服务方式协作。为了避免破坏已有运行路径，本次没有移动核心入口文件，而是新增根文档进行整理：

- `README.md`：项目入口说明、启动方式、文件用途。
- `PROJECT_STATUS.md`：当前进度、完成能力、未完成工作。

建议下一阶段上线前再做一次结构化拆分，例如把长文件拆为：

- `server/`：数据库、API 路由、鉴权、服务封装。
- `client/`：页面渲染、状态管理、语音录制、UI 工具函数。
- `docs/`：部署、接口、测试、产品路线。

当前先保持零依赖、少文件、可快速部署的形态。

## 已完成

### 产品闭环

- 首页 Dashboard：今日待办、今日完成、明日计划、进行中事项、项目进展、快速记录入口和项目快录入口。
- 快速记录页：自然语言输入、语音按钮、智能整理、结构化结果编辑、保存。
- 整理结果预览：只保留已完成、进行中、待办事项。
- 报告中心：生成类型只保留周报、月报；今日日报放在今日结构化记录中自动生成。
- 待办管理：新增、编辑、删除、完成、状态/优先级/日期筛选，默认显示未开始事项。
- 项目归档：首页项目快录、项目开始日期、收尾日期、关闭归档、自动生成项目复盘。
- 复盘与洞察：按日期、关键词、项目筛选，展开查看记录和总结，并支持删除记录或总结。
- 个人定制：岗位、目标、汇报对象、输出风格、模板、关键词、协作对象配置。
- 个人账号：邮箱 + 密码注册登录；每个用户独立 SQLite 数据库。

### 技术闭环

- 静态页面由 Node 内置 HTTP 服务托管。
- SQLite 数据库已落地，认证库包含 `users`；个人业务库包括 `work_records`、`tasks`、`projects`、`summaries`、`user_preferences`。
- 前端数据源已从 `localStorage` 切换为 `/api/*`。
- 旧 `data/app.sqlite` 数据支持首个注册用户继承。
- AI 服务统一封装在 `aiService.js`，包含记录整理、语音文本润色、日报、周报、月报、项目复盘生成；提示词已强化准确转译、项目关联和避免误分类。
- DeepSeek API 通过环境变量接入。
- 百度语音识别通过环境变量接入，密钥不进入前端响应和代码。
- 个人登录已替代访问口令成为主要访问保护。
- 备份脚本已支持认证库和用户库，默认保留最近 14 天备份。
- 部署说明已覆盖云服务器、systemd、Caddy HTTPS、SQLite 备份。
